# <a name="manage-log-analytics-using-azure-resource-manager-templates"></a>Zarządzanie przy użyciu szablonów Menedżera zasobów Azure analizy dziennika

Możesz użyć [szablony Menedżera zasobów Azure] (.. / azure-resource-manager/resource-group-authoring-templates.md) tworzenie i konfigurowanie obszarów roboczych analizy dziennika. Przykłady zadania, które można wykonywać za pomocą szablonów:

+ Tworzenie obszaru roboczego
+ Dodaj rozwiązanie
+ Tworzenie zapisane wyszukiwania
+ Tworzenie grupy komputera
+ Włącz zbieranie dzienników programu IIS z komputerów ze zainstalowany agent systemu Windows
+ Zbieranie liczników wydajności z systemu Windows i Linux oraz komputerów
+ Zbierać zdarzenia z syslog na komputerach Linux 
+ Zbierać zdarzenia z dzienniki zdarzeń systemu Windows
+ Zbieranie dzienników niestandardowe
+ Dodawanie agenta analizy dziennika Azure maszyn wirtualnych
+ Konfigurowanie analizy dziennika zindeksować danych zebranych za pomocą diagnostyki Azure


Ten artykuł zawiera szablon przykładów pokazujących, niektóre konfiguracji, które można wykonywać przy użyciu szablonów.

## <a name="create-and-configure-a-log-analytics-workspace"></a>Tworzenie i konfigurowanie obszaru roboczego analizy dziennika

Poniższy przykładowy szablon pokazano, jak:

1.  Tworzenie obszaru roboczego
2.  Dodawanie rozwiązania do obszaru roboczego
3.  Tworzenie zapisane wyszukiwania
4.  Tworzenie grupy komputera
5.  Włącz zbieranie dzienników programu IIS z komputerów ze zainstalowany agent systemu Windows
6.  Zbieranie liczników wydajności dysku logicznego z komputerów Linux (% używane Inodes; Bezpłatne megabajtów; % Używane miejsca; Przeniesienia dysku/s; Odczyt dysku/s; Zapisy dysku/s)
7.  Zbierać zdarzenia syslog z komputerów Linux
8.  Zbieranie zdarzeń błąd i ostrzeżenie w dzienniku zdarzeń aplikacji na komputerach z systemem Windows
9.  Zbieranie licznika dostępne MB pamięci na komputerach z systemem Windows
10. Zbieranie dziennika niestandardowego 
11. Zbieranie dzienników programu IIS i dzienniki zdarzeń systemu Windows, napisane przez Azure diagnostyki na koncie miejsca do magazynowania

### <a name="deploying-the-sample-template"></a>Wdrażanie Poniższy przykładowy szablon

Aby wdrożyć Poniższy przykładowy szablon:

1. Zapisywanie załączonych próbki w pliku, na przykład`azuredeploy.json` 
2. Edytowanie szablonu konfiguracji, które mają
3. Wdrażanie szablonu za pomocą programu PowerShell lub wiersz polecenia

#### <a name="powershell"></a>Programu PowerShell

`New-AzureRmResourceGroupDeployment -Name <deployment-name> -ResourceGroupName <resource-group-name> -TemplateFile azuredeploy.json`

#### <a name="command-line"></a>Wiersz polecenia

```
azure config mode arm
azure group deployment create <my-resource-group> <my-deployment-name> --TemplateFile azuredeploy.json
```
